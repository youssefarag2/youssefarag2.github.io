Image Placeholders Needed

To complete your portfolio website, you'll need the following images:

  • hero-bg.jpg - A background image for the hero section. Recommended size: 1920x1080px. This could be a subtle tech-related image or a clean abstract background.
  • profile.jpg - Your professional photo for the About section. Recommended size: 500x500px. A professional headshot is ideal.
  • favicon.png - A small icon for browser tabs. Recommended size: 32x32px.
  • project-placeholder.jpg - A fallback image for projects without specific images. Recommended size: 600x400px.
  • Project-specific images:
    • rag-ai-chatbot.jpg - Image for RAG AI Chatbot project
    • travel-track.jpg - Image for Travel Track project
    • taskify.jpg - Image for Taskify project
    • deleviro.jpg - Image for Deleviro project

Note: Until you add these images, the website will display fallback content. The website is fully functional without these images, but adding them will enhance the visual appeal.

You can use sites like Unsplash for free high-quality images or create your own.
